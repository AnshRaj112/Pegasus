Pegasus validation engine must now support huge unsorted CSV reconciliation for files up to 100GB without crashing memory.

The current in-memory dataframe join architecture is failing and must be redesigned into a partitioned external-memory reconciliation engine.

Requirements:

Core Requirements:

- Do NOT load full files into memory
- Support unsorted CSV files
- Support files up to 100GB
- Use streaming/chunk-based processing
- Keep memory bounded and predictable

Required Architecture:

1. UID Generation

- Generate deterministic SHA256 UID hashes
- Support composite UID columns
- Normalize null values safely

2. Hash Partitioning Engine

- Partition rows using:
  partition_id = hash(uid) % N
- Ensure identical UIDs always land in the same partition
- Write partitions incrementally to disk
- Avoid keeping partitions in memory

3. Streaming Reader

- Read CSVs in chunks
- Configurable chunk sizes
- Support configurable delimiters and encodings
- Use generators/iterators

4. Partition Comparator

- Compare source and target partitions independently
- Detect:
  - missing rows
  - extra rows
  - column mismatches

5. Spill-To-Disk Architecture

- Use temp files for partitions
- Add automatic cleanup
- Avoid RAM-heavy operations

6. Mismatch Writer

- Write mismatches incrementally
- Avoid collecting all mismatches in memory
- Support JSONL or CSV append-based writing

7. Reconciliation Coordinator

- Manage:
  - chunk processing
  - partition routing
  - comparison lifecycle
  - cleanup lifecycle

Technical Constraints:

- Use Polars only for chunk-level operations
- Avoid full dataframe joins
- Avoid full file reads
- Add memory-safe logging
- Add type hints
- Add docstrings
- Use clean modular architecture

Modules to create:

- StreamCSVReader
- UIDGenerator
- PartitionManager
- PartitionWriter
- PartitionComparator
- MismatchWriter
- ReconciliationCoordinator

Do NOT implement Spark, Kubernetes, Beam, or distributed workers yet.

Focus ONLY on making huge unsorted CSV validation stable and memory-efficient.
