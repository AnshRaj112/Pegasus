## How to Generate

```python
from pathlib import Path
from pegasus.validation.engine import run_tabular_file_pair
from pegasus.validation.pipeline import TabularPipelineConfig

run_tabular_file_pair(
    Path("/data/source.csv"),
    Path("/data/target.csv"),
    identity_columns=["id"],
    config=TabularPipelineConfig(
        partition_preset="large",
        record_key_columns=["id"],
        compare_columns=["id", "amount", "status"],
    ),
    report_path=Path("docs/VALIDATION_RESULTS.md"),
)
```

For Kubernetes workers, set `PEGASUS_WORKER_INDEX` / `PEGASUS_WORKER_COUNT` (or `TabularPipelineConfig` partition range). Each job also writes `tabular_pipeline_report.md` under the job directory when using the integrated path.

All full tabular validation jobs use the six-stage pipeline; see `tabular_pipeline_report.md` in the job directory.
